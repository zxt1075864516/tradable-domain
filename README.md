# tradable-domain

域名出售展示页
本项目旨在为域名提供一个简洁、高效的展示平台，向潜在买家清晰地传达域名的可售状态及联系方式。通过静态页面实现快速部署，确保全球用户都能随时访问并获取购买信息。
核心功能
可售状态展示：明确标识当前域名处于“出售中”状态，吸引潜在买家注意。
所有者信息公示：提供域名所有人的联系方式，方便买家直接沟通。
响应式设计：适配桌面端与移动端设备，确保在不同屏幕尺寸下均有良好的浏览体验。
技术实现
项目采用轻量级前端技术栈，确保加载速度与兼容性：
HTML5：构建页面基础结构。
CSS3：实现简洁美观的样式布局。
JavaScript：处理简单的交互逻辑（如复制联系方式功能）。
文件结构
```text
tradable-domain/
├── index.html      # 主页面，展示域名出售信息
├── css/            # 样式文件目录
└── js/             # 脚本文件目录
```
